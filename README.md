# :house: MacTime
Mac address changer every n` seconds


# :bulb: Installing
`git clone https://github.com/Xttiriey/MacTime.git`

`cd MacTime`

`sudo pip install -r requirements.txt`

# :computer: Using
`sudo python macTime.py -i wlan0 -t 10`
  
`-t`  time interval in seconds
  
`-i` your interface
